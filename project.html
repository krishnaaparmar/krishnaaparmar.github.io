Project Name : NumbersGame

Logo: None

Project Description:

NumbersGame is a mobile application for iOS devices where number display has a specific number as a button and a specific functionality. Each number pressed, if less than 5 will be incremented upto the highest number displayed, if is equal to five then will not change, and if greater than 5 then will be decremented untill it reaches 5. Upon reset it will be set to its initial value.
